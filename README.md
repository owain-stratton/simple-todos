## Meteor Simple-Todos Application

```
# Clone this GitHub repo
git clone https://github.com/owain-stratton/simple-todos.git
cd simple-todos

#install meteor
curl https://install.meteor.com/ | sh

# install project dependencies
npm install

# run project (runs at localhost:3000)
meteor

# run tests (runs at localhost:9090)
meteor test --port 9090 --driver-package practicalmeteor:mocha
```
